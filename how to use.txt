Windows setup

1. Open a terminal in the project folder.

Example:
~\Python-Exam-Agent

2. Enter the backend folder.

Command:
cd backend

3. Check the Python version.

Command:
python --version

4. Create a virtual environment.

Command:
python -m venv .venv

5. Activate the virtual environment.

Command:
.venv\Scripts\activate

6. Install backend dependencies.

Command:
pip install -r requirements.txt

7. Start the backend server.

Command:
uvicorn app.main:app --reload

8. Open a second terminal in the project folder.

Example:
~\Python-Exam-Agent

9. Start the frontend server.

Command:
cd frontend
python -m http.server 5173

10. Open the frontend in your browser.

URL:
http://127.0.0.1:5173

Exam flow

1. Click Start Exam.
2. Choose Full exam: all 8 questions.
3. Click Start Timed Exam.
4. Complete the exam within 90 minutes.
5. Write code in the editor.
6. Click Compile & Run Tests.
7. Review visible test results and hidden backend edge test status.
8. Click Finish Exam to see the final score.

Generate full exam example

{
  "topic": "__all__",
  "difficulty": "standard",
  "number_of_questions": 8,
  "username": "test"
}

Notes:
- The default test user is created automatically when the backend starts.
- Questions are grouped by topic/type, not by easy, medium, or hard.
- Use "standard" as the difficulty value.
- Use "__all__" as the topic value to start the full 8-question exam.
- The frontend exam timer is 90 minutes.
- The final score is shown as 1/8, 2/8, and so on. A full score shows PASS.
- Some backend edge test cases are hidden, so students cannot see every test input.
- Current topics include lists, integers, number theory, file processing, matrices, patterns, and sets and words.
- Some questions use temporary files such as CSV files or dictionary.txt during automatic grading.
- The local SQLite database is created automatically when the backend starts.
